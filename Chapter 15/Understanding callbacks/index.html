Callbacks

  • A callback is a function.
  • It gets passed as an argument to another function.
  • The main function executes its task.
  • Once the main function completes its task, it calls (or "callbacks to") the provided callback function to execute additional code.

    • When the main task is completed, it uses the callback function to notify you and execute any additional code you've provided in the callback. This way, you can coordinate and sequence tasks that depend on each other or handle asynchronous operations effectively.
